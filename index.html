Recommended Steps to Become a Good Python Programmer

  1. Learn basic Python
  2. Install an application called 'Visual Studio Code'
  3. Learn about git and github
  4. Install git on your computer
  5. Create a github account
  6. Create a simple python program, then use git to keep track of its versions (git init)
  7. Make a change in your simple python program, then add the changes (git add . ) and commit the changes (git commit -m"any comment")
  8. Make random changes to your simple python program, then use git restore to go back to the last saved version of your program
  9. Install pip env, learn what it is and how to use it (pipenv shell, pip install)
  10. Install Flask, using pipenv shell, then pipenv install, then learn how Flask works
  11. Install a Python module of your choice using pipenv
  12. Learn how to create 2 python files and how to import functions from file2.py into file1.py
  13. Learn how to publish python / Flask powered websites using pythonanywhere.com
  14. Learn how to use AI tools such as ChatGPT, Google Bard, Phinder to learn how to code faster
  15. Register on stackoverflow.com and find out how to make use of it
  16. Do the CS50 course from Harvard on Edex (it's for free!)
  17. Learn about Object Oriented Programming in Python (OOP)
  18. Learn about Session variables in Flask
  19. Learn the basics of HTML and CSS
  20. Learn the basics of JavaScript






These recommended steps will take years to take. One step can be done faster then another.
