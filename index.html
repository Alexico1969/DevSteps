Recommended Steps to Become a Good Python Programmer (and beyond)

  1. Learn basic Python
  2. Install an application called 'Visual Studio Code'
  3. Install Python on your local machine (Windows: ADD TO PATH!!)
  4. Create a basic Python file in Visual Studio Code and run it
  5. Learn about git and github
  6. Install git on your computer
  7. Create a github account
  8. Learn how to use the 'TERMINAL' in VS Code
  9. Learn how to use git in VS Code
  10. Learn how to push and pull to and from GitHub
  11. Install pip env, learn what it is and how to use it (pipenv shell, pip install)
  12. Install Flask, using pipenv shell, then pipenv install, then learn how Flask works
  13. Learn how to create a basic Flask website
  14. Install a Python module of your choice using pipenv
  15. Learn how to create 2 python files and how to import functions from file2.py into file1.py
  16. Learn how to publish python / Flask powered websites using pythonanywhere.com
  17. Learn how to use AI tools such as ChatGPT, Google Bard, Phinder to learn how to code faster
  18. Register on stackoverflow.com and find out how to make use of it
  19. Learn the basics of SQL (recommended module: sqlite3)
  20. Do the CS50 course from Harvard on Edex (it's for free!)
  21. Learn about Object Oriented Programming in Python (OOP)
  22. Learn about Session variables in Flask
  23. Learn the basics of HTML and CSS
  24. Learn the basics of JavaScript
  25. Learn the basics of JQuery
  26. Learn the basics of Bootstrap
  27. Learn the basics of AJAX
  28. Learn the basics of JSON
  29. Learn the basics of XML
  30. Learn the basics of APIs
  31. Create a Flask website, using API's
  32. Create a Flask website, using JavaScript, JQuery and Bootstrap in the templates
  33. Learn the basics of Django
  34. Create a simple Django website
  35. Learn the basics of Angular
  36. Learn the basics of React
  37. Learn the basics of Vue
  38. Learn the basics of Node.js
  39. Learn the basics of MongoDB
  40. Learn the basics of MySQL
  41. Learn the basics of PostgreSQL
  42. Learn the basics of PHP
  43. Learn the basics of C#
  44. Learn the basics of Java
  45. Learn the basics of C++






These recommended steps will take years if you want to take them all. Some steps take less time than others.
